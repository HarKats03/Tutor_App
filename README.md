Tutor App 🎓

A comprehensive student and income management application for tutors, built with Python and CustomTkinter.

🚀 Features

  - Modern UI: Built with CustomTkinter for a sleek and functional user interface.
  - Data Management: All data is stored securely in a local SQLite database.
  - Excel Reports: Automatically generate Excel reports (.xlsx) to track income and student progress.
  - Cross-platform: Fully compatible with Windows and macOS.

🛠️ Installation

1.  Clone the repository: git clone 
2.  Install the required dependencies: pip install -r requirements.txt

💻 Usage

Run the application: python tutor_app.py

📦 Project Structure

  - tutor_app.py: Main application logic.
  - config.toml: Configuration settings.
  - tutor_manager.db: SQLite database file.
  - logo.png / logo.ico: Application assets.
  - tutor_app.spec: PyInstaller configuration for creating executables.
